COMMAND 1: CHECK INVENTORY
---------------------------
Function: Displays the current number of beer crates in inventory

Accepted Phrases:
  - "check inventory"
  - "status"
  - "how many" (e.g., "how many beers", "how many beers left")
  - "inventory"
  - "beers left"
  - "beer count"

Example Usage:
  User: check inventory
  Bot: üç∫ Current inventory: 50 crates

  User: status
  Bot: üç∫ Current inventory: 50 crates

Response Format:
  Shows current crate count with beer emoji indicator


COMMAND 2: ADD CRATES TO INVENTORY
-----------------------------------
Function: Adds full crates directly to inventory.

Accepted Phrases:
  - "add X crates" (e.g., "add 1 crate", "add 5 crates")
  - "added X crates"
  - "add X crates to inventory"
  - "added X crates to inventory"
  - "stock X crates"
  - "restock X crates"
  - "received X crates"

Example Usage:
  User: add 2 crates
  Bot: Added 2 crates to inventory. Total: 52 crates.

  User: add 1 crate
  Bot: Added 1 crate to inventory. Total: 51 crates.

Note: 
  - X must be a positive number
  - Adds full crates directly (no conversion needed)
  - Standard crate size: 24 beers per crate


COMMAND 3: SUBTRACT CRATES FROM INVENTORY
------------------------------------------
Function: Removes full crates from inventory.

Accepted Phrases:
  - "subtract X crates"
  - "removed X crates"
  - "take X crates"
  - "remove X crates"
  - "subtract X crates from inventory"
  - "removed X crates from inventory"

Example Usage:
  User: subtract 2 crates
  Bot: Removed 2 crates from inventory. 48 crates remaining.

  User: remove 1 crate
  Bot: Removed 1 crate from inventory. 49 crates remaining.

  User: subtract 50 crates
  Bot: Removed 50 crates from inventory. Inventory is now empty.

Note: 
  - X must be a positive number
  - Removes full crates directly
  - Inventory cannot go below 0 crates
  - Standard crate size: 24 beers per crate

TECHNICAL DETAILS
-----------------
- All commands are case-insensitive
- The bot uses natural language processing
- Commands work with partial phrases (e.g., "how many" triggers inventory check)
- Inventory is stored in inventory.json file (crates only)
- Default starting inventory: 50 crates (configurable in .env file)
- All operations work with full crates only
- Standard crate size: 24 beers per crate
